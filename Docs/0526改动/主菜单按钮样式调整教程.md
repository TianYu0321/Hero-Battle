# 主菜单按钮样式调整教程（Godot 4）

## 一、让 Godot 识别新图片

你刚放进 `assets/ui/main_menu/` 的图，Godot 还没导入。

1. 打开 Godot 编辑器
2. 左侧 **文件系统** 面板展开 `assets/ui/main_menu/`
3. 看到 `btn_normal.png`、`btn_hover.png` 等文件
4. **按 Ctrl+S 保存项目**（触发自动导入）
5. 如果图片还是灰的，**项目 → 重新加载当前项目**（或关闭重开）

导入成功后，图片会显示缩略图。

---

## 二、打开 Theme 资源

1. 在 **文件系统** 找到 `resources/themes/menu_theme.tres`
2. **双击打开**
3. 看右侧 **Inspector** 面板

你会看到类似这样的结构：
```
Button
  └── styles
      ├── disabled: StyleBoxTexture
      ├── focus: StyleBoxTexture
      ├── hover: StyleBoxTexture
      ├── normal: StyleBoxTexture
      └── pressed: StyleBoxTexture
```

---

## 三、调整按钮边框拉伸（核心）

我们的按钮图是**异形边框**，四角有菱形装饰，中间要拉伸。为了不让四角变形，需要设置 **Texture Margin**。

### 操作步骤：

1. 在 Inspector 里点击 `Button/styles/normal`
   - 展开后显示一个 `StyleBoxTexture` 资源
2. 点击这个 `StyleBoxTexture` 资源（会展开更多属性）
3. 找到 **`Texture Margin`** 区域，有 4 个值：
   - **Left / Right**：左右两边不拉伸的宽度（保护菱形装饰）
   - **Top / Bottom**：上下两边不拉伸的高度（保护上下边框）

### 默认值我已经设好了：
```
Left = 160, Right = 160
Top = 100, Bottom = 100
```

### 怎么判断对不对？

**看场景预览**：
- 如果按钮四角被**压扁或拉伸变形** → Margin 太小，调大
- 如果按钮中间内容区域太小、文字被挤到两边 → Margin 太大，调小

**推荐调法**：
1. 打开 `scenes/main_menu/menu.tscn`
2. 选中任意一个按钮（如 `BtnNewGame`）
3. 看右上角 **2D 视图**，按钮会实时显示当前样式
4. 切回 `menu_theme.tres`，改一个 Margin 数字，立刻看 2D 视图变化

## 四、对比两套发光 Hover

现在有两套 hover：
- `btn_hover.png`（默认发光）
- `btn_hover2.png`（第二套发光）

### 全局切换（所有按钮统一）：

1. 打开 `menu_theme.tres`
2. 找到 `Button/styles/hover`
3. 展开 `StyleBoxTexture`
4. 找到 **`Texture`** 属性
5. 把 `btn_hover.png` 换成 `btn_hover2.png`
   - 点击 Texture 右边的资源路径
   - 选择 `btn_hover2.png`
6. **Ctrl+S 保存 Theme**

### 单独给某个按钮试效果：

1. 打开 `scenes/main_menu/menu.tscn`
2. 选中你想试的按钮（比如 `BtnSettings`）
3. Inspector 拉到最下面，找到 **Theme Overrides**
4. 展开 **Styles**
5. 在 `Hover` 那一行，点击 `<空>`
6. 选择 **新建 StyleBoxTexture**
7. 展开这个新建的 StyleBoxTexture
8. **Texture** 属性选择 `btn_hover2.png`
9. **Texture Margin** 填：
   ```
   Left=160, Right=160, Top=100, Bottom=100
   ```
10. 运行场景，鼠标悬停到这个按钮上看效果

## 五、调文字颜色

1. 打开 `menu_theme.tres`
2. 找到 `Button/colors/font_color`
3. 点击颜色框，选你喜欢的金色/米白色
4. 推荐值：
   - 米金色：`#F2D9A0`（RGB: 242, 217, 160）
   - 浅金色：`#FFE4B5`（RGB: 255, 228, 181）
   - 当前值：`#F2D999`（我已经调好的）

## 六、调背景图和人物层级

### 换背景图

背景图路径固定在这里：
```
assets/ui/main_menu/background.png
```

直接覆盖这个文件，Godot 会自动刷新。**不需要改任何代码**。

### 人物层级大小/位置

1. 打开 `scenes/main_menu/menu.gd`
2. 找到 `_setup_character_layer()` 函数（文件最末尾附近）
3. 改这几行：

```gdscript
# 人物在屏幕上的位置（当前是居中偏下）
sprite.position = Vector2(960, 540)

# 缩放比例（按屏幕高度适配）
# 改这个 1080.0 可以调人物大小
var scale_needed := 1080.0 / person_tex.get_height()
```

比如人物太大了，改成：
```gdscript
var scale_needed := 800.0 / person_tex.get_height()
```

或者人物位置要往左移：
```gdscript
sprite.position = Vector2(700, 600)
```

## 七、常见问题

### Q：按钮显示为全白或一片黑？
A：图片还没导入成功。**保存项目 → 重启编辑器**。

### Q：按钮边框被拉伸变形了？
A：**Texture Margin** 太小。左右 Margin 调大 20~50，试到四角恢复正常为止。

### Q：按钮文字被边框挡住了？
A：调 `Content Margin`（在 StyleBoxTexture 里）。把 `Content Margin Left` 从默认的 12 改成 **60 或 80**，给文字留出左边距。

### Q：怎么只改一个按钮，不改全局？
A：用 **Theme Overrides**（见第四节"单独给某个按钮试效果"）。
