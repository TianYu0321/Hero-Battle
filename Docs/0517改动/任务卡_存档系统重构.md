# 任务卡：存档系统重构（独立数据层）

## 背景

**当前问题**：
- `_save_at_floor_entrance()` 直接调用 `_run.to_dict()` + `_hero.to_dict()`，字段名与 RuntimeRun/RuntimeHero 强耦合
- `continue_from_save()` 用 `RunSnapshot.from_dict()` 解析，字段名映射不一致（如 `current_floor` vs `current_turn`）
- 修改运行时对象字段会破坏旧存档兼容性，没有版本号管理
- 存档数据里混入了 UI 相关数据（如 `node_options`），但缺少关键运行时状态（如 `special_floor_phase`）

**目标**：
- 定义独立的 `RunSaveData` 数据层，与 RuntimeRun/RuntimeHero 完全解耦
- 存档结构带版本号，支持旧存档迁移
- 只存"恢复一局游戏所需的纯数据"，不存 UI 状态、不存运行时对象引用
- `continue_from_save()` 和 `_save_at_floor_entrance()` 基于同一套字段名，不再有二义性

---

## 存档数据结构设计

```
RunSaveData (Dictionary)
├── version: int              # 存档格式版本，当前 = 1
├── saved_at: int             # unix timestamp
├── current_floor: int        # 当前层数（1~30），对应 RuntimeRun.current_turn
├── hero:
│   ├── config_id: int
│   ├── current_hp: int
│   ├── max_hp: int
│   ├── vit: int
│   ├── str: int
│   ├── agi: int
│   ├── tec: int
│   ├── mnd: int
│   └── training_counts: Dictionary  # {"vit": 3, "str": 1, ...}
├── gold: int
├── battle_win_count: int
├── elite_win_count: int
├── shop_visit_count: int
├── pvp_10th_result: int      # 0=未触发, 1=胜, 2=败
├── pvp_20th_result: int
├── node_history: Array[Dictionary]   # 历史节点记录（精简）
├── partners: Array[Dictionary]     # 已招募伙伴快照
├── node_options: Array[Dictionary] # 当前层选项（SL不变）
├── event_forecast_charges: int
├── run_seed: int
└── run_status: int           # 1=进行中, 2=胜利, 3=失败, 4=放弃
```

**注意**：不存以下数据
- 运行时对象引用（`RuntimeRun`、`RuntimeHero` 实例）
- UI 面板状态（`shop_panel.visible` 等）
- Tween/Timer 状态
- 动画播放状态
- `BattlePlaybackRecorder` 事件流（太大且不必须）

## 执行步骤

### Step 1：新建 `RunSaveData` 辅助类（或纯函数）

**文件**：`scripts/systems/run_save_data.gd`（新建）

**职责**：把 RuntimeRun/RuntimeHero/伙伴列表 转换为存档字典，以及反向恢复。不与运行时对象耦合。

```gdscript
class_name RunSaveData
extends RefCounted

const CURRENT_VERSION: int = 1

static func from_runtime(run: RuntimeRun, hero: RuntimeHero, partners: Array, node_options: Array, forecast_charges: int) -> Dictionary:
    var data := {
        "version": CURRENT_VERSION,
        "saved_at": int(Time.get_unix_time_from_system()),
        "current_floor": run.current_turn,
        "hero": {
            "config_id": run.hero_config_id,
            "current_hp": hero.current_hp,
            "max_hp": hero.max_hp,
            "vit": hero.current_vit,
            "str": hero.current_str,
            "agi": hero.current_agi,
            "tec": hero.current_tec,
            "mnd": hero.current_mnd,
            "training_counts": hero.training_counts.duplicate(),
        },
        "gold": run.gold_owned,
        "battle_win_count": run.battle_win_count,
        "elite_win_count": run.elite_win_count,
        "shop_visit_count": run.shop_visit_count,
        "pvp_10th_result": run.pvp_10th_result,
        "pvp_20th_result": run.pvp_20th_result,
        "node_history": _compress_history(run.node_history),
        "partners": _partners_to_dicts(partners),
        "node_options": node_options.duplicate(true),
        "event_forecast_charges": forecast_charges,
        "run_seed": run.run_seed,
        "run_status": run.run_status,
    }
    return data

static func to_runtime(data: Dictionary, character_manager: CharacterManager) -> Dictionary:
    ## 返回：{"run": RuntimeRun, "hero": RuntimeHero, "partners": Array[RuntimePartner], "forecast_charges": int}
    ## 由 RunController 调用后自行赋值
    
    # 版本迁移
    var version: int = data.get("version", 0)
    if version == 0:
        data = _migrate_v0_to_v1(data)
    
    var hero_data: Dictionary = data.get("hero", {})
    var hero := RuntimeHero.new()
    hero.hero_config_id = hero_data.get("config_id", 0)
    hero.current_hp = hero_data.get("current_hp", 100)
    hero.max_hp = hero_data.get("max_hp", 100)
    hero.current_vit = hero_data.get("vit", 10)
    hero.current_str = hero_data.get("str", 10)
    hero.current_agi = hero_data.get("agi", 10)
    hero.current_tec = hero_data.get("tec", 10)
    hero.current_mnd = hero_data.get("mnd", 10)
    hero.training_counts = hero_data.get("training_counts", {})
    
    var run := RuntimeRun.new()
    run.hero_config_id = hero.hero_config_id
    run.current_turn = data.get("current_floor", 1)
    run.gold_owned = data.get("gold", 0)
    run.battle_win_count = data.get("battle_win_count", 0)
    run.elite_win_count = data.get("elite_win_count", 0)
    run.shop_visit_count = data.get("shop_visit_count", 0)
    run.pvp_10th_result = data.get("pvp_10th_result", 0)
    run.pvp_20th_result = data.get("pvp_20th_result", 0)
    run.node_history = data.get("node_history", [])
    run.run_seed = data.get("run_seed", randi())
    run.run_status = data.get("run_status", 1)
    
    var partners: Array = []
    var partner_dicts: Array = data.get("partners", [])
    for pd in partner_dicts:
        var p = RuntimePartner.from_dict(pd)
        partners.append(p)
    
    return {
        "run": run,
        "hero": hero,
        "partners": partners,
        "node_options": data.get("node_options", []),
        "event_forecast_charges": data.get("event_forecast_charges", 0),
    }

static func _migrate_v0_to_v1(data: Dictionary) -> Dictionary:
    ## v0 是旧的直接 to_dict() 格式，字段名混乱
    var migrated := data.duplicate(true)
    
    # 楼层：旧版可能叫 current_turn
    if not migrated.has("current_floor") and migrated.has("current_turn"):
        migrated["current_floor"] = migrated["current_turn"]
    
    # 英雄：旧版 hero 字段结构不同
    var old_hero = migrated.get("hero", {})
    if old_hero is Dictionary:
        var new_hero := {}
        new_hero["config_id"] = old_hero.get("hero_config_id", old_hero.get("config_id", 0))
        new_hero["current_hp"] = old_hero.get("current_hp", 100)
        new_hero["max_hp"] = old_hero.get("max_hp", 100)
        new_hero["vit"] = old_hero.get("current_vit", old_hero.get("vit", 10))
        new_hero["str"] = old_hero.get("current_str", old_hero.get("str", 10))
        new_hero["agi"] = old_hero.get("current_agi", old_hero.get("agi", 10))
        new_hero["tec"] = old_hero.get("current_tec", old_hero.get("tec", 10))
        new_hero["mnd"] = old_hero.get("current_mnd", old_hero.get("mnd", 10))
        new_hero["training_counts"] = old_hero.get("training_counts", {})
        migrated["hero"] = new_hero
    
    migrated["version"] = 1
    return migrated

static func _compress_history(history: Array) -> Array:
    ## 压缩历史记录，只保留必要字段，减少存档体积
    var compressed: Array = []
    for h in history:
        compressed.append({
            "turn": h.get("turn", 0),
            "node_type": h.get("node_type", 0),
        })
    return compressed

static func _partners_to_dicts(partners: Array) -> Array:
    var result: Array = []
    for p in partners:
        if p is RuntimePartner:
            result.append(p.to_dict())
        elif p is Dictionary:
            result.append(p.duplicate())
    return result
```

**注意**：
- `RuntimePartner.from_dict()` 需要你确认已有这个方法，如果没有需要补
- `RuntimeHero.training_counts` 字段需要你确认存在，如果不存在用空字典兜底

**修改后**：
```gdscript
func continue_from_save(save_data: Dictionary) -> bool:
    print("[RunController] 恢复存档")
    if save_data.is_empty():
        push_error("[RunController] Cannot continue from empty save data")
        return false
    
    var result: Dictionary = RunSaveData.to_runtime(save_data, _character_manager)
    if result.is_empty():
        push_error("[RunController] Failed to restore save data")
        return false
    
    _run = result.get("run") as RuntimeRun
    _hero = result.get("hero") as RuntimeHero
    var partners: Array = result.get("partners", [])
    var node_options: Array = result.get("node_options", [])
    var forecast_charges: int = result.get("event_forecast_charges", 0)
    
    # 恢复伙伴到 CharacterManager
    _character_manager.clear_partners()
    for p in partners:
        if p is RuntimePartner:
            _character_manager.add_partner_runtime(p)
    
    # 恢复节点池（确保 SL 后选项不变）
    _current_node_options = node_options.duplicate(true)
    
    # 恢复事件透视次数
    if forecast_charges > 0:
        var forecast_system: EventForecastSystem = get_node_or_null("EventForecastSystem")
        if forecast_system != null:
            forecast_system.set_charges(forecast_charges)
    
    # 恢复状态
    _state = RunState.RUNNING_NODE_SELECT
    _special_floor_phase = SpecialFloorPhase.NONE
    _battle_result_phase = BattleResultPhase.NONE
    
    # 发射信号恢复 UI
    EventBus.emit_signal("run_started", {
        "hero_id": _run.hero_config_id,
        "partner_ids": [],
        "run_seed": _run.run_seed,
    })
    EventBus.emit_signal("gold_changed", _run.gold_owned, 0, "continue_from_save")
    EventBus.emit_signal("run_continued", _run.current_turn)
    
    # 如果存了当前层选项，直接恢复；否则重新生成
    if not _current_node_options.is_empty():
        EventBus.emit_signal("floor_changed", _run.current_turn, _MAX_TURNS, _get_phase_name())
        EventBus.emit_signal("node_options_presented", _current_node_options)
        print("[RunController] 恢复存档选项: %d 个, 当前层=%d" % [_current_node_options.size(), _run.current_turn])
    else:
        _change_state(RunState.RUNNING_NODE_SELECT)
    
    print("[RunController] 存档恢复完成，当前层=%d" % _run.current_turn)
    return true
```

### Step 6：主菜单 "继续游戏" 流程确认

**文件**：`scenes/main_menu/menu.gd`（或对应主菜单逻辑）

**确认**：
1. 点击"继续游戏"时，从 `SaveManager.load_run_state()` 读取存档

## 与战斗系统统一任务的关联

战斗系统统一后，存档数据里不再需要保存 `BattlePlaybackRecorder`（本来也没存）。统一后所有战斗都从 BattleEngine 实时计算 + recorder 回放，存档只需要保存"打到第几层、什么属性、多少金币"这些静态数据。两份任务卡互不阻塞，可并行实施。
