# 任务卡：主菜单背景动态效果（下雨 + 人物呼吸）

## 目标

用 Godot 代码让静态背景 PNG 动起来，不动原图。当前项目 1280x720，背景图 1920x1080 自适应缩放。

效果清单：
1. **下雨** — 主氛围，暗蓝灰色雨滴斜向下飘落
2. **底部光带脉动** — 保留，暗金光带微弱明暗
3. **镜头呼吸** — 背景轻微缩放，6 秒周期
4. **人物呼吸** — 后续角色立绘上下浮动 + 微弱缩放

---

## 各节点属性配置

### BackgroundTexture
- `layout_mode = 1` (Anchors)
- `anchors_preset = 15` (Full Rect)
- `expand_mode = IGNORE_SIZE`
- `stretch_mode = KEEP_ASPECT_CENTERED`

### FloorLightOverlay
- `anchors_preset = 7` (Bottom Wide)
- `anchor_top = 0.75`
- `color = Color(0.90, 0.75, 0.35, 0.05)`  `#E6C040`

## 动态效果实现

### 1. 下雨效果（主氛围）

**方式**：GPUParticles2D，两层叠加 —— 近景大雨滴 + 远景小雨滴

文件：`scenes/main_menu/menu.gd`

```gdscript
@onready var rain_parent: Node2D = $BackgroundLayer/RainParent

func _start_rain() -> void:
    ## 第一层：近景大雨滴（速度较快，较粗，较明显）
    var rain_near := GPUParticles2D.new()
    rain_near.name = "RainNear"
    rain_near.position = Vector2(960, -100)          ## 顶部中央偏上，覆盖全宽
    rain_near.amount = 300
    rain_near.lifetime = 1.2
    rain_near.preprocess = 2.0
    rain_near.visibility_rect = Rect2(-200, -200, 2240, 1480)
    
    var mat_near := ParticleProcessMaterial.new()
    mat_near.emission_shape = ParticleProcessMaterial.EMISSION_SHAPE_BOX
    mat_near.emission_box_extents = Vector3(1200, 10, 1)
    mat_near.direction = Vector3(0.15, 1, 0)         ## 微微向右倾斜
    mat_near.spread = 2.0
    mat_near.initial_velocity_min = 500.0
    mat_near.initial_velocity_max = 700.0
    mat_near.gravity = Vector3(0, 800, 0)
    mat_near.scale_min = 1.0
    mat_near.scale_max = 2.0
    mat_near.color = Color(0.55, 0.65, 0.75, 0.35)  ## 暗蓝灰，半透明
    rain_near.process_material = mat_near
    
    ## 雨滴纹理：细长线条
    var drop_texture_near := GradientTexture2D.new()
    drop_texture_near.gradient = Gradient.new()
    drop_texture_near.gradient.colors = [Color(1,1,1,0.6), Color(1,1,1,0)]
    drop_texture_near.width = 2
    drop_texture_near.height = 24
    drop_texture_near.fill_from = Vector2(0.5, 0)
    drop_texture_near.fill_to = Vector2(0.5, 1)
    rain_near.texture = drop_texture_near
    
    rain_parent.add_child(rain_near)
    
    ## 第二层：远景小雨滴（更细更淡，速度较慢）
    var rain_far := GPUParticles2D.new()
    rain_far.name = "RainFar"
    rain_far.position = Vector2(960, -50)
    rain_far.amount = 400
    rain_far.lifetime = 2.0
    rain_far.preprocess = 3.0
    rain_far.visibility_rect = Rect2(-200, -200, 2240, 1480)
    
    var mat_far := ParticleProcessMaterial.new()
    mat_far.emission_shape = ParticleProcessMaterial.EMISSION_SHAPE_BOX
    mat_far.emission_box_extents = Vector3(1400, 10, 1)
    mat_far.direction = Vector3(0.1, 1, 0)
    mat_far.spread = 5.0
    mat_far.initial_velocity_min = 300.0
    mat_far.initial_velocity_max = 450.0
    mat_far.gravity = Vector3(0, 500, 0)
    mat_far.scale_min = 0.5
    mat_far.scale_max = 1.0
    mat_far.color = Color(0.45, 0.50, 0.60, 0.15)   ## 更淡
    rain_far.process_material = mat_far
    
    var drop_texture_far := GradientTexture2D.new()
    drop_texture_far.gradient = Gradient.new()
    drop_texture_far.gradient.colors = [Color(1,1,1,0.3), Color(1,1,1,0)]
    drop_texture_far.width = 1
    drop_texture_far.height = 16
    drop_texture_far.fill_from = Vector2(0.5, 0)
    drop_texture_far.fill_to = Vector2(0.5, 1)
    rain_far.texture = drop_texture_far
    
    rain_parent.add_child(rain_far)
```

**关键参数说明**：
- `direction.x = 0.15` — 雨滴微微向右斜，有风感
- `gravity.y = 800` — 加速下落，自然
- `visibility_rect` — 必须覆盖全屏，否则粒子被 Godot 裁剪掉
- `preprocess = 2.0/3.0` — 场景加载时提前模拟几秒，避免开局空屏
- 两层叠加比单层更有纵深感

### 2. 底部光带微弱脉动（保留）

func _start_floor_pulse() -> void:
    var tween := create_tween().set_loops()
    tween.tween_property(floor_light, "modulate:a", 0.15, 3.0)
    tween.tween_property(floor_light, "modulate:a", 0.04, 3.0)

### 3. 镜头呼吸（背景轻微缩放）

```gdscript
@onready var bg_texture: TextureRect = $BackgroundLayer/BackgroundTexture

func _start_camera_breath() -> void:
    bg_texture.pivot_offset = Vector2(960, 540)    ## 1920x1080 中心
    var tween := create_tween().set_loops()
    tween.tween_property(bg_texture, "scale", Vector2(1.02, 1.02), 6.0)
    tween.tween_property(bg_texture, "scale", Vector2(1.0, 1.0), 6.0)
```

### 4. 人物呼吸（角色立绘）

```gdscript
@onready var character_layer: CanvasLayer = $CharacterLayer

func _add_character_with_breath(sprite_path: String, pos: Vector2) -> void:
    var sprite := Sprite2D.new()
    sprite.texture = load(sprite_path)
    sprite.position = pos
    sprite.scale = Vector2(0.8, 0.8)
    sprite.name = "CharacterSprite"
    character_layer.add_child(sprite)
    
    ## 呼吸 1：上下浮动 6px，4秒周期
    var float_tween := create_tween().set_loops()
    float_tween.tween_property(sprite, "position:y", pos.y - 6, 2.0)
    float_tween.tween_property(sprite, "position:y", pos.y + 6, 2.0)
    
    ## 呼吸 2：微弱缩放 0.98↔1.02，6秒周期（和镜头呼吸错开相位）
    var scale_tween := create_tween().set_loops()
    scale_tween.tween_property(sprite, "scale", Vector2(0.82, 0.82), 3.0).set_delay(1.0)
    scale_tween.tween_property(sprite, "scale", Vector2(0.78, 0.78), 3.0)

## 后续调用：
## _add_character_with_breath("res://assets/characters/hero_menu.png", Vector2(1400, 540))
```

**角色位置建议**：
- 1920x1080 坐标：`Vector2(1400, 540)` — 画面右侧偏中，和左侧菜单按钮形成对角平衡
- 1280x720 下会自动缩放，视觉上位置比例一致
- 立绘底部对齐地面光带，有"站在斗技场入口"的感觉

## 可选增强

### 雨滴打在地面的微弱反光

在 `FloorLightOverlay` 下方加一个 `RainSplash` 粒子层：

```gdscript
func _start_rain_splash() -> void:
    var splash := GPUParticles2D.new()
    splash.name = "RainSplash"
    splash.position = Vector2(960, 950)         ## 靠近底部地面
    splash.amount = 60
    splash.lifetime = 0.3
    splash.preprocess = 2.0
    
    var mat := ParticleProcessMaterial.new()
    mat.emission_shape = ParticleProcessMaterial.EMISSION_SHAPE_BOX
    mat.emission_box_extents = Vector3(1000, 20, 1)
    mat.direction = Vector3(0, -1, 0)
    mat.spread = 30.0
    mat.initial_velocity_min = 20.0
    mat.initial_velocity_max = 40.0
    mat.gravity = Vector3(0, -50, 0)
    mat.scale_min = 0.5
    mat.scale_max = 1.5
    mat.color = Color(0.55, 0.65, 0.75, 0.2)
    splash.process_material = mat
    
    var splash_texture := GradientTexture2D.new()
    splash_texture.gradient = Gradient.new()
    splash_texture.gradient.colors = [Color(1,1,1,0.4), Color(1,1,1,0)]
    splash_texture.width = 4
    splash_texture.height = 4
    splash.texture = splash_texture
    
    rain_parent.add_child(splash)
```

效果：地面偶尔溅起微小水花反光，增加真实感。粒子量小，不抢眼。

## 分辨率处理

- 当前项目 1280x720，背景图 1920x1080
- `BackgroundTexture` 用 `KEEP_ASPECT_CENTERED` + `IGNORE_SIZE`
- 1920x1080 图缩放到 1280x720 时裁掉左右边缘（废墟部分），主视觉穹顶和光带完整保留
- 雨滴粒子发射位置按 1920x1080 写（`Vector2(960, -100)`），因为粒子世界坐标不随 UI 缩放，但实际覆盖范围用 `visibility_rect` 保证
- 切 1920x1080 时背景图自动撑满，雨滴密度和位置正确，无需改动
