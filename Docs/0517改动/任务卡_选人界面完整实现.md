# 任务卡：选人界面完整实现

## 目标

主菜单点击"开始游戏"后进入的选人界面。延续暗黑霓虹风格，左侧纵向英雄列表 + 右侧大立绘 + 底部属性预览。支持已解锁/未解锁状态、新手/老手入口分流。

---

## 设计约束

- 分辨率：当前 1280x720（出图按 1920x1080，Godot KEEP_ASPECT_CENTERED 缩放）
- 风格：暗黑霓虹（深黑底 + 暗金 #E6C040 强调 + 红 #D93826 危险/锁定 + 蓝 #5A8FD0 可用）
- 字体：标题 Cinzel，数值 Oxanium，中文 NotoSansSC
- 延续主菜单下雨氛围（可选：用雨幕粒子覆盖整个选人界面，或淡入淡出过渡）

## 各节点属性

### BackgroundTexture
- `anchors_preset = 15` (Full Rect)
- `expand_mode = IGNORE_SIZE`
- `stretch_mode = KEEP_ASPECT_CENTERED`
- `modulate = Color(0.4, 0.4, 0.5, 1.0)` — 比主菜单更暗，突出前景

### HeroCard_x（每个英雄卡片）
- 尺寸：宽 280，高 72
- 默认态：暗底 #1A1A1F，1px 边框 #333338
- Hover 态：边框渐变为 #5A8FD0（蓝），0.2 秒
- 选中态：左边框 4px #E6C040（暗金），背景 #252530
- 锁定态：整体灰度 0.5，右上角小锁图标
- 内容：左侧 48x48 头像缩略图 + 右侧名称+定位

### HeroBigPortrait
- 尺寸：800x600（实际显示区域）
- 位置：RightPanel 中央偏上
- 未选中英雄时：显示剪影占位图 + "???"
- 切换英雄时：旧立绘 fade_out (0.15s) → 新立绘 fade_in (0.15s)，无位置移动

### LockOverlay
- `anchors_preset = 15` (Full Rect)
- `color = Color(0, 0, 0, 0.6)`
- 中间：锁图标 TextureRect（64x64，灰色 #888）
- 下方：Label "完成前置挑战解锁" 或具体解锁条件

### StatPreviewContainer（五维）
- 每个属性：Label（Vit/Str/Agi/Tec/Mnd）+ ProgressBar（宽 80 高 8）+ 数值 Label
- ProgressBar 颜色按属性：Vit=绿 #4ECDC4，Str=红 #FF6B6B，Agi=黄 #E6C040，Tec=蓝 #5A8FD0，Mnd=紫 #9B59B6
- 数值 Label 用 Oxanium，右对齐

### StartRunButton
- 默认：暗底 #1A1A1F，边框 1px #E6C040，文字暗金
- Hover：背景渐变（底→顶）#252530 → #3A3020，边框 2px #E6C040
- 点击：缩放 0.95 + AudioManager.play_ui("confirm")
- 未选中英雄时：disabled，透明度 0.4
- 选中锁定英雄时：disabled，文字变 "未解锁"

### 2. 立绘切换
```gdscript
func _show_hero_portrait(hero_config_id: int) -> void:
    var tween_out := create_tween()
    tween_out.tween_property(hero_big_portrait, "modulate:a", 0.0, 0.12)
    await tween_out.finished
    
    var path: String = ConfigManager.get_hero_portrait_path(hero_config_id)
    if not path.is_empty():
        hero_big_portrait.texture = load(path)
    
    var tween_in := create_tween()
    tween_in.tween_property(hero_big_portrait, "modulate:a", 1.0, 0.18)
```

### 3. 五维条动画
```gdscript
func _animate_stat_bars(hero_config_id: int) -> void:
    var cfg: Dictionary = ConfigManager.get_hero_config(hero_config_id)
    var stats: Dictionary = cfg.get("base_stats", {})
    
    var vit_bar: ProgressBar = $BottomPanel/StatPreviewContainer/VitBar
    var target_vit: float = stats.get("vit", 10) / 20.0  ## 假设 20 是满值
    
    var tween := create_tween().set_parallel()
    tween.tween_property(vit_bar, "value", target_vit * 100, 0.4).set_ease(Tween.EASE_OUT).set_trans(Tween.TRANS_QUAD)
    ## ... 其他四属性同理
```

### 4. 切场景过渡
```gdscript
func _start_transition_to_run() -> void:
    transition_overlay.visible = true
    transition_overlay.modulate.a = 0.0
    
    var tween := create_tween()
    tween.tween_property(transition_overlay, "modulate:a", 1.0, 0.6)
    await tween.finished
    
    ## 传递参数给 RunMain
    GameManager.start_new_run(_selected_hero_id, _is_veteran_mode)
    ## 或 EventBus.emit_signal("start_run_requested", ...)
    
    get_tree().change_scene_to_file("res://scenes/run_main/run_main.tscn")
```

```gdscript
    pass

static func get_hero_portrait_path(hero_config_id: int) -> String:
    var cfg := get_hero_select_config(hero_config_id)
    return cfg.get("portrait_path", "")

static func is_hero_unlocked(hero_config_id: int) -> bool:
    var cfg := get_hero_select_config(hero_config_id)
    var condition: String = cfg.get("unlock_condition", "")
    if condition.is_empty():
        return true
    ## 查 player_data 解锁记录
    var player_data: Dictionary = SaveManager.load_player_data()
    var unlocked: Array = player_data.get("unlocked_heroes", [1])  ## 默认只有 hero_id=1 解锁
    return hero_config_id in unlocked
```

## 新手 / 老手分流

| 模式 | 标识 | 差异 |
|------|------|------|
| 新手 | `is_veteran = false` | 敌人 HP-20%，商店降价 30%，首次救援必出 S 级伙伴，前 5 层可预览敌人信息 |
| 老手 | `is_veteran = true` | 标准数值，商店原价，救援随机，敌人信息始终 ??? |

**实现**：`DifficultyToggle` 点击切换，视觉状态更新，`_is_veteran_mode` bool 传入 `GameManager.start_new_run()`。
