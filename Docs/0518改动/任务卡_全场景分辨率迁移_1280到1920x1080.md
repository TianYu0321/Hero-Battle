# 任务卡：全场景分辨率迁移 1280x720 → 1920x1080

## 背景

当前项目以 1280x720 为基准开发，大量 UI 使用硬编码坐标（offset/position）和 `Vector2(1280, 720)`。迁移到 1920x1080（比例 1.5x）后，所有写死坐标的元素会错位、飞出屏幕或挤在一角。

**迁移原则**：
- **首选**：改用 Anchor + Container 相对布局，自适应任何分辨率
- **次选**：必须硬编码的（粒子发射点、特效中心、Tween 目标位置）乘以 1.5
- **项目设置**：Window Size 改 1920x1080，Stretch 用 `canvas_items` + `keep`

---

## Step 1：项目设置修改

**路径**：Godot Editor → Project → Project Settings → Display → Window

| 设置项 | 当前值 | 修改后 |
|--------|--------|--------|
| Size → Width | 1280 | **1920** |
| Size → Height | 720 | **1080** |
| Stretch → Mode | (默认或 viewport) | **canvas_items** |
| Stretch → Aspect | (默认) | **keep** |

**文件**：`project.godot`（ Godot 会自动写，也可以手动改）
```
[display]
window/size/viewport_width=1920
window/size/viewport_height=1080
window/stretch/mode=canvas_items
window/stretch/aspect=keep
```

**说明**：`canvas_items` 模式 UI 按设计分辨率渲染，然后缩放到实际窗口。1920x1080 成为"设计分辨率"，实际窗口可以更小（如 1280x720），Godot 自动缩放 UI。这让你可以在大分辨率下开发，同时兼容小屏。

### 2.2 `scenes/main_menu/menu.gd` — 特效位置硬编码

```gdscript
# 当前（1280 基准）
_left_hand_glow.position = Vector2(253, 480)
_left_hand_particles.position = Vector2(253, 480)
_fog_sprite.position = Vector2(640, 360)
_tower_glow.position = Vector2(920, 147)  # 等同

# 修改方式 A：直接乘以 1.5
_left_hand_glow.position = Vector2(380, 720)
# ...

# 修改方式 B（推荐）：改为相对于 viewport 的动态计算
var vp_size := get_viewport().get_visible_rect().size
_left_hand_glow.position = Vector2(vp_size.x * 0.198, vp_size.y * 0.667)  # 380/1920, 720/1080
```

**建议**：`menu_fx_config.json` 里的 `position` 也改为比例值 `[0.198, 0.667]`，代码里运行时乘 `vp_size`。这样任何分辨率下特效位置都对。

## Step 4：迁移执行策略

### 策略 A：快速适配（1.5x 乘法）

适合赶时间、场景少的情况。

1. 改 `project.godot` 窗口尺寸为 1920x1080
2. 所有 `offset_*` 和 `position` 乘以 1.5
3. 所有 `Vector2(1280, 720)` 改为 `Vector2(1920, 1080)`
4. 运行测试

**缺点**：以后切其他分辨率（如 2560x1440）还要再改一遍。

### 策略 B：Anchor/Container 重构（推荐）

适合长期维护。

1. 改 `project.godot` 窗口尺寸为 1920x1080
2. 每个场景里：
   - 根 Control 用 `Full Rect`
   - 顶部标题 → `anchors_preset = 5` (Center Top) + `anchor_top` 百分比
   - 底部按钮区 → `anchors_preset = 7` (Bottom Wide) 或 `anchors_preset = 3` (Bottom Right)
   - 居中弹窗 → `anchors_preset = 8` (Center) + 固定 `custom_minimum_size`
   - 弹窗内部 → VBoxContainer / HBoxContainer / GridContainer
3. 代码里的粒子位置改为 `get_viewport().size` 比例
4. 删除所有不再需要的 offset 值

**优点**：以后切任何分辨率都不需要改坐标。
