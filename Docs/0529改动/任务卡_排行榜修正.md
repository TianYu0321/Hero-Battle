# 任务卡：排行榜修正（返回键 + 净胜场排序 + PVPManager 数据）

## 问题

1. `menu.gd` 中调用 `LeaderboardSystem.new()` 获取排行榜，但该类排序依据为 `total_score`，项目要求为 **净胜场（net_wins）**
2. `menu.gd` 未连接 `leaderboard_panel.closed` 信号，面板弹出后**无法通过返回键关闭**
3. `leaderboard_panel.gd` 的 `show_rankings` 字段名与 `PVPManager.get_leaderboard()` 返回格式不一致

---

## Step 1：menu.gd — 改用 PVPManager 数据 + 连接关闭信号

修改 `_on_leaderboard_pressed`：

```gdscript
func _on_leaderboard_pressed() -> void:
    print("[MainMenu] 【点击】排行榜")
    
    ## 使用 PVPManager 获取按净胜场排序的排行榜
    var rankings: Array[Dictionary] = PVPManager.get_leaderboard()
    
    if _leaderboard_panel != null:
        ## 连接关闭信号（如未连接）
        if not _leaderboard_panel.closed.is_connected(_on_leaderboard_closed):
            _leaderboard_panel.closed.connect(_on_leaderboard_closed)
        
        _leaderboard_panel.show_rankings(rankings)
    else:
        push_warning("[MainMenu] LeaderboardPanel 未找到")

func _on_leaderboard_closed() -> void:
    print("[MainMenu] 排行榜关闭")
    ## 面板已自行隐藏，此处无需额外操作
```

---

## Step 2：leaderboard_panel.gd — 适配 PVPManager 数据格式

PVPManager.get_leaderboard() 返回格式：
```gdscript
{
    "rank": int,        # 排名
    "name": String,     # 玩家名
    "net_wins": int,    # 净胜场
    "wins": int,        # 总胜场
    "losses": int,      # 总败场
    "is_player": bool,  # 是否当前玩家
}
```

修改 `show_rankings`：

```gdscript
func show_rankings(rankings: Array[Dictionary]) -> void:
    ## 清空旧列表
    for child in ranking_container.get_children():
        child.queue_free()
    
    ## 生成列表项
    for entry in rankings:
        var row = preload("res://scenes/leaderboard/ranking_row.tscn").instantiate()
        
        ## 字段名适配
        var rank: int = entry.get("rank", 0)
        var name: String = entry.get("name", entry.get("hero_name", entry.get("player_name", "???")))
        var net_wins: int = entry.get("net_wins", 0)
        var wins: int = entry.get("wins", 0)
        var losses: int = entry.get("losses", 0)
        var is_player: bool = entry.get("is_player", false)
        
        row.setup(rank, name, net_wins, wins, losses, is_player)
        ranking_container.add_child(row)
    
    visible = true
```

## Step 3：ranking_row.gd — 扩展 setup 方法

如 `ranking_row.gd` 的 `setup` 方法签名不匹配，修改为：

```gdscript
func setup(rank: int, player_name: String, net_wins: int, wins: int = 0, losses: int = 0, is_player: bool = false) -> void:
    ## 排名
    var rank_label: Label = $RankLabel
    rank_label.text = str(rank)
    
    ## 玩家名（当前玩家高亮）
    var name_label: Label = $NameLabel
    name_label.text = player_name
    if is_player:
        name_label.add_theme_color_override("font_color", Color(0.25, 0.55, 0.9, 1))  ## 蓝色高亮
    else:
        name_label.add_theme_color_override("font_color", Color(0.25, 0.25, 0.28, 1))
    
    ## 净胜场
    var net_label: Label = $NetWinsLabel
    net_label.text = str(net_wins)
    
    ## 胜/败场（可选显示）
    var detail_label: Label = $DetailLabel  ## 如有此节点
    if detail_label != null:
        detail_label.text = "%d胜 / %d败" % [wins, losses]
```

## Step 4：leaderboard_panel.tscn — 确保 CloseButton 存在

确认场景中有 `CloseButton` 节点：

```
[node name="CloseButton" type="Button" parent="."]
layout_mode = 2
text = "返回"
```

如缺少，在 `leaderboard_panel.gd` 的 `_ready` 中动态创建：

```gdscript
func _ready() -> void:
    visible = false
    
    ## 确保关闭按钮存在
    if close_button == null:
        close_button = Button.new()
        close_button.text = "返回"
        close_button.custom_minimum_size = Vector2(120, 44)
        add_child(close_button)
    
    close_button.pressed.connect(_on_close)
```
