## Style 01：卷轴羊皮纸血条 (Parchment Scroll Bar)

### 视觉描述
- **外框**：深木色圆角矩形，3px边框，带内阴影营造凹陷感
- **底色**：羊皮纸纹理色 `#F4E8D0`，带有细微的暗角
- **填充**：从左到右由暗红 `#8B0000` → 鲜红 `#DC143C` 渐变，低血量(<30%)时变为金色闪烁
- **文字**：HP数值使用手写体风格，置于血条右端
- **特效**：血量减少时填充边缘有轻微"撕纸"抖动；低于30%时整个血条微微呼吸发光

### 适用场景
- 与当前项目的木质/羊皮纸UI风格**完美契合**
- 适合作为默认血条，无需额外素材

### 预览代码
```gdscript
# 使用 StyleBoxFlat 即可实现，无需外部素材
var style_bg := StyleBoxFlat.new()
style_bg.bg_color = Color(0.35, 0.25, 0.15)  # 深木底
style_bg.border_color = Color(0.2, 0.12, 0.06)
style_bg.border_width_left = 3
style_bg.border_width_top = 3
style_bg.border_width_right = 3
style_bg.border_width_bottom = 3
style_bg.corner_radius_top_left = 6
style_bg.corner_radius_top_right = 6
style_bg.corner_radius_bottom_left = 6
style_bg.corner_radius_bottom_right = 6
style_bg.shadow_color = Color(0, 0, 0, 0.4)
style_bg.shadow_size = 4

var style_fill := StyleBoxFlat.new()
style_fill.bg_color = Color(0.86, 0.08, 0.24)  # 暗红
style_fill.corner_radius_top_left = 3
style_fill.corner_radius_top_right = 3
style_fill.corner_radius_bottom_left = 3
style_fill.corner_radius_bottom_right = 3
```

## Style 02：经典JRPG分段血条 (Segmented JRPG Bar)

### 一句话解释
**把血条切成像电池电量一样的小格子，一格一格地掉血，而不是平滑滑动。**

### 视觉描述（通俗版）
想象你的手机电量显示：它不是一根平滑的进度条，而是由10个小格子组成。
- **满血时**：10格全是亮的翡翠绿色
- **受攻击时**：一次掉2格，那2格先闪一下白光，然后"啪"地碎裂消失
- **半血时**：还剩5格，颜色变成琥珀黄色（警告色）
- **红血时**：只剩2~3格，变成珊瑚红色，并且整排格子节奏性地闪烁
- **空血时**：所有格子熄灭，只剩暗色的格子轮廓

每格之间有明显的深色分隔线，像一节节电池或一格格血瓶。

### 专业描述
- **结构**：血条被均匀分为5~10段，每段之间有2px的深色分隔线
- **填充**：已填充的段为亮绿色/蓝色，未填充段为深灰黑色
- **阈值变色**：>70%为翡翠绿，30%~70%为琥珀黄，<30%为珊瑚红并伴随段闪烁
- **文字**：段数可替代数字显示（如5格=50HP），也可保留数字在左侧
- **特效**：每损失一段，该段有0.1s的"碎裂"缩小动画后才熄灭

### 来源参考
- 《宝可梦》系列对战HP条（最经典的分段血条）
- 《火焰纹章：风花雪月》
- 《歧路旅人/Octopath Traveler》

### 为什么适合敌人？
敌人的分段血条能让玩家**一眼判断战况**：
- "还剩8格，稳了" → 绿色
- "只剩4格了，进入斩杀线" → 黄色
- "最后2格，快死了" → 红色闪烁

这种"数格子"的直觉比看百分比数字更快，特别适合回合制策略游戏。

### 预览代码
```gdscript
# 使用 TextureProgress 或 多个 TextureRect 拼接实现
# 每一段是一个独立的 ProgressBar，宽度固定
const SEGMENTS := 10
const SEGMENT_WIDTH := 16
const SEGMENT_GAP := 2

# 颜色阈值
var COLOR_HIGH := Color(0.18, 0.76, 0.49)   # 翡翠绿
var COLOR_MID := Color(0.95, 0.65, 0.15)    # 琥珀黄
var COLOR_LOW := Color(0.85, 0.22, 0.15)    # 珊瑚红
```

## Style 03：灵魂心形容器 (Soul Heart Containers)

### 视觉描述
- **结构**：一排心形容器（♥），通常每颗心代表固定HP（如20HP）
- **状态**：满血=红心，半血=半颗心（左半边红右半边灰），空血=灰/黑心
- **边框**：每颗心有金色描边，增加精致感
- **布局**：可分两行显示，避免过长
- **特效**：受伤时对应的心有一个"碎裂"粒子效果；回复时心有"粘合"发光效果

### 来源参考
- 《塞尔达传说》系列
- 《死亡细胞》
- 《以撒的结合》

### 适用场景
- **SD纸片剧场风格的绝佳搭档**，心形本身就很可爱/冒险感
- 适合主角HP展示，敌人可以保留传统血条形成对比

### 素材需求
- 需要3张心形纹理：满、半、空（48x48或64x64）
- 或直接用 Godot `Polygon2D` / `StyleBoxFlat` 绘制心形

### 预览代码
```gdscript
# 使用 GridContainer + TextureRect 排列心形
const HEART_VALUE := 20  # 每颗心代表的HP
const MAX_HEARTS := 10

# 动态计算需要显示的心
func update_hearts(current_hp: int, max_hp: int) -> void:
    var total_hearts := ceili(float(max_hp) / HEART_VALUE)
    for i in range(total_hearts):
        var heart_hp := mini(HEART_VALUE, maxi(0, current_hp - i * HEART_VALUE))
        var texture := _heart_full if heart_hp == HEART_VALUE else (_heart_half if heart_hp > 0 else _heart_empty)
        _heart_slots[i].texture = texture
```

## Style 04：剧场幕布丝带 (Stage Curtain Ribbon)

### 视觉描述
- **主题契合度最高**：专为"SD纸片剧场"设计
- **外框**：顶部有微型红色幕布褶皱装饰（用三角形或纹理模拟）
- **主体**：一条细长的丝带，颜色为酒红 `#722F37`，填充部分为亮红丝带质感
- **褶皱感**：血条表面有轻微的波浪线纹理（可用Shader或重复图案模拟）
- **装饰**：血条两端有金色的"穗子"或流苏装饰
- **特效**：血量变化时，丝带像被拉动一样有轻微的弹性抖动

### 适用场景
- **本项目的主题血条**，与"剧场"概念完美呼应
- 可放置在角色卡片顶部或独立在HP区域

### 素材需求
- 幕布顶部装饰纹理（可代码绘制）
- 可选：丝带褶皱Shader

### 预览代码
```gdscript
# 使用 StyleBoxFlat + 顶部装饰 Polygon2D
var ribbon_bg := StyleBoxFlat.new()
ribbon_bg.bg_color = Color(0.25, 0.08, 0.08)  # 深酒红底
ribbon_bg.border_color = Color(0.9, 0.75, 0.35) # 金边
ribbon_bg.border_width_left = 1
ribbon_bg.border_width_top = 1
ribbon_bg.border_width_right = 1
ribbon_bg.border_width_bottom = 1

# 幕布褶皱（用 Polygon2D 在血条上方绘制）
var curtain := Polygon2D.new()
curtain.color = Color(0.55, 0.1, 0.1)
curtain.polygon = PackedVector2Array([
    Vector2(0, 0), Vector2(8, -6), Vector2(16, 0),
    Vector2(24, -6), Vector2(32, 0), Vector2(40, -6), Vector2(48, 0)
])
```

## Style 05：符咒封印条 (Talisman Seal Bar)

### 视觉描述
- **外框**：符咒形状，上下边缘不是直的，而是像撕开的纸边（锯齿状或波浪状）
- **底色**：泛黄的符纸色 `#E8DCC8`，边缘有朱红色符文装饰（可用小图标重复排列）
- **填充**：填充色为朱红 `#B91D2C`，带有"墨水晕染"效果
- **低血量**：<30%时符咒边缘开始变黑烧焦，填充色变为暗紫并伴随微小火焰粒子
- **文字**：HP数值用竖排或仿古字体，置于血条中央

### 来源参考
- 《阴阳师》战斗UI
- 《东方Project》符卡系统
- 《鬼灭之刃》手游

### 适用场景
- 日式奇幻冒险感极强
- 适合Boss战或特殊敌人，与主角的普通血条形成差异

### 素材需求
- 符咒边框纹理（可用StyleBoxFlat的锯齿border模拟）
- 可选：符文小图标（16x16重复平铺）

### 预览代码
```gdscript
# 使用自定义 _draw() 绘制符咒形状
func _draw() -> void:
    var rect := Rect2(Vector2.ZERO, size)
    # 底色
    draw_rect(rect, Color(0.91, 0.86, 0.78))
    # 朱红填充（根据value比例）
    var fill_width := size.x * (value / max_value)
    draw_rect(Rect2(Vector2.ZERO, Vector2(fill_width, size.y)), Color(0.73, 0.11, 0.17))
    # 边框锯齿（用draw_line绘制小段波浪线）
    for x in range(0, int(size.x), 8):
        draw_line(Vector2(x, 0), Vector2(x+4, -3), Color(0.73, 0.11, 0.17), 1.5)
        draw_line(Vector2(x+4, -3), Vector2(x+8, 0), Color(0.73, 0.11, 0.17), 1.5)
```

## Style 06：卡牌镶嵌条 (Card Inlay Bar)

### 视觉描述
- **设计理念**：血条不是独立的，而是**镶嵌在角色卡片底部**
- **外框**：金属质感细边框（银/铜色），2px，圆角与角色卡片统一
- **底色**：半透明深灰（不遮挡卡片背景）
- **填充**：宝石质感渐变——翡翠绿（高血量）→ 琥珀（中血量）→ 红宝石（低血量）
- **光泽**：填充部分有模拟高光的白线（顶部1px，alpha=0.3）
- **文字**：HP数字使用等宽字体，置于填充色上方（确保对比度）

### 来源参考
- 《游戏王：大师决斗》卡片UI
- 《影之诗》随从面板
- 《炉石传说》随从血条

### 适用场景
- **与当前角色卡片展示方式完美契合**
- 如果采用"文档中的卡片化布局"，这是最佳选择
- 即使保持当前Sprite2D立绘，也可在卡片底部镶嵌

### 预览代码
```gdscript
# 金属边框StyleBox
var metal_bg := StyleBoxFlat.new()

metal_bg.border_color = Color(0.65, 0.55, 0.4)  # 古铜金
metal_bg.border_width_left = 2
metal_bg.border_width_top = 2
metal_bg.border_width_right = 2
metal_bg.border_width_bottom = 2
metal_bg.corner_radius_bottom_left = 8
metal_bg.corner_radius_bottom_right = 8

# 宝石填充（带高光）
var gem_fill := StyleBoxFlat.new()
gem_fill.bg_color = Color(0.18, 0.76, 0.49)  # 翡翠绿，按血量变色
gem_fill.corner_radius_bottom_left = 6
gem_fill.corner_radius_bottom_right = 6
```

## Style 07：像素复古条 (Pixel Retro Bar)

### 视觉描述
- **边缘**：绝对硬边，无抗锯齿，无圆角，纯像素风格
- **外框**：1px黑色边框，内部1px白色高光（顶部和左侧）营造立体感
- **填充**：纯色块，>50%亮绿 `#4EEE94`，25%~50%黄 `#FFD700`，<25%红 `#FF4500`
- **心跳**：低血量时，整个血条有节奏地缩放（1.0 → 1.05 → 1.0，0.8s周期）
- **文字**：像素字体，紧挨血条右侧或嵌入内部

### 来源参考
- 《八方旅人/Octopath Traveler》
- 《LIVE A LIVE》
- 《星之海/Sea of Stars》

### 适用场景
- 如果项目中角色立绘是像素风格，这是最佳选择
- 但当前项目是SD纸片剧场（Sprite2D但非像素），契合度中等
- 可作为"特殊模式"或"回忆关卡"的血条

### 预览代码
```gdscript
# 关键：关闭抗锯齿，硬边缘
var pixel_bg := StyleBoxFlat.new()
pixel_bg.bg_color = Color(0.15, 0.15, 0.15)
pixel_bg.border_color = Color(0.0, 0.0, 0.0)
pixel_bg.border_width_left = 1
pixel_bg.border_width_top = 1
pixel_bg.border_width_right = 1
pixel_bg.border_width_bottom = 1
pixel_bg.corner_radius_top_left = 0  # 无圆角 = 像素感
pixel_bg.corner_radius_top_right = 0
pixel_bg.corner_radius_bottom_left = 0
pixel_bg.corner_radius_bottom_right = 0
pixel_bg.anti_aliasing = false       # 关闭抗锯齿

# 低血量心跳动画
func _low_hp_pulse() -> void:
    var tween := create_tween().set_loops()
    tween.tween_property(self, "scale", Vector2(1.05, 1.05), 0.4)
    tween.tween_property(self, "scale", Vector2.ONE, 0.4)
```
