# 任务卡：营救UI + 爬塔UI 伙伴改为卡片图片

## 背景

当前两处伙伴展示都是文字为主：
- **营救UI**：RescuePopup 中用 Label 显示伙伴名字和描述，无头像
- **爬塔UI**：PartnerHUD 中虽然有 Avatar TextureRect，但整体是条状布局，不是卡片感

**目标**：两处都用带头像的卡片形式展示伙伴，图已备好。

---

## 子任务 1：营救UI（RescuePopup）卡片化

### 当前代码

`scenes/rescue/rescue_popup.gd` → `_fill_rescue_option()`：

### 修改方案

每个候选伙伴一个卡片，横向排列（3个卡片一排）：

```
RescuePopup
├── TitleLabel ("选择你的伙伴")
├── CandidatesContainer (HBoxContainer, 居中, 间距 24)
│   ├── PartnerCard_0 (PanelContainer, 200x280)
│   │   ├── CardBorder (ColorRect)           ← 稀有度色边框
│   │   ├── Portrait (TextureRect, 160x160)  ← 伙伴头像
│   │   ├── RarityBadge (Label)              ← S/A/B/C 角标
│   │   ├── NameLabel (Label)                ← 名字
│   │   ├── RoleLabel (Label)                ← 职业/定位
│   │   └── SelectButton (Button)            ← "招募"按钮
│   ├── PartnerCard_1
│   └── PartnerCard_2
```

### rescue_popup.gd 修改

```gdscript
## @onready 新增
@onready var candidates_container: HBoxContainer = $CandidatesContainer

func show_popup(candidates: Array) -> void:
    ## 清空旧卡片
    for child in candidates_container.get_children():
        child.queue_free()
    
    ## 创建3张卡片
    for i in range(candidates.size()):
        var card := _create_partner_card(candidates[i], i)
        candidates_container.add_child(card)
    
    visible = true
    AudioManager.play_ui("popup_open")

func _create_partner_card(candidate: Dictionary, index: int) -> PanelContainer:
    var card := PanelContainer.new()
    card.custom_minimum_size = Vector2(200, 280)
    card.name = "PartnerCard_%d" % index
    
    ## 边框样式按稀有度
    var rarity: String = candidate.get("rarity", "C")
    var border_color := _get_rarity_color(rarity)
    var style := StyleBoxFlat.new()
    style.bg_color = Color(0.08, 0.08, 0.10, 0.95)
    style.border_color = border_color
    style.border_width_left = 3
    style.border_width_top = 3
    style.border_width_right = 3
    style.border_width_bottom = 3
    style.corner_radius_top_left = 8
    style.corner_radius_top_right = 8
    style.corner_radius_bottom_left = 8
    style.corner_radius_bottom_right = 8
    card.add_theme_stylebox_override("panel", style)
    
    var vbox := VBoxContainer.new()
    vbox.alignment = BoxContainer.ALIGNMENT_CENTER
    vbox.add_theme_constant_override("separation", 8)
    card.add_child(vbox)
    
    ## 稀有度角标（右上角）
    var badge := Label.new()
    badge.text = rarity
    badge.horizontal_alignment = HORIZONTAL_ALIGNMENT_CENTER
    badge.add_theme_font_size_override("font_size", 14)
    badge.add_theme_color_override("font_color", border_color)
    vbox.add_child(badge)
    
    ## 头像
    var portrait := TextureRect.new()
    portrait.custom_minimum_size = Vector2(160, 160)
    portrait.expand_mode = TextureRect.EXPAND_FIT_WIDTH_PROPORTIONAL
    portrait.stretch_mode = TextureRect.STRETCH_KEEP_ASPECT_CENTERED
    var portrait_path: String = candidate.get("portrait_path", "")
    if portrait_path.is_empty():
        portrait_path = ConfigManager.get_partner_portrait_path(candidate.get("partner_id", 0))
    if not portrait_path.is_empty():
        portrait.texture = load(portrait_path)
    vbox.add_child(portrait)
    
    ## 名字
    var name_label := Label.new()
    name_label.text = candidate.get("name", "???")
    name_label.horizontal_alignment = HORIZONTAL_ALIGNMENT_CENTER
    name_label.add_theme_font_size_override("font_size", 16)
    name_label.add_theme_color_override("font_color", Color("#E6C040"))
    vbox.add_child(name_label)
    
    ## 职业/定位
    var role_label := Label.new()
    role_label.text = candidate.get("role", "伙伴")
    role_label.horizontal_alignment = HORIZONTAL_ALIGNMENT_CENTER
    role_label.add_theme_font_size_override("font_size", 12)
    role_label.add_theme_color_override("font_color", Color("#888888"))
    vbox.add_child(role_label)
    
    ## 招募按钮
    var btn := Button.new()
    btn.text = "招募"
    btn.custom_minimum_size = Vector2(120, 36)
    btn.pressed.connect(func(): _on_rescue_selected(index))
    vbox.add_child(btn)
    
    return card

func _get_rarity_color(rarity: String) -> Color:
    match rarity:
        "S": return Color("#E6C040")
        "A": return Color("#5A8FD0")
        "B": return Color("#4ECDC4")
        _:   return Color("#888888")
```

## 子任务 2：爬塔UI（RunMain PartnerHUD）卡片化

### 当前代码

`run_main.gd` → `_fill_partner_slot()`：
- 已有 Avatar(TextureRect, 32x32)、NameLabel、LevelLabel、SkillGauge
- 布局是横向 HBoxContainer（头像+文字+充能条），不是卡片感

### 修改方案

改为小型竖版卡片（类似营救卡片的缩小版），竖向排列在左侧：

```
PartnerPanel (VBoxContainer, 左侧)
├── PartnerTitle ("随行伙伴")
├── PartnerCardSlot_0 (PanelContainer, 140x180)   ← 小型卡片
│   ├── CardBorder (StyleBoxFlat)
│   ├── Portrait (TextureRect, 80x80)
│   ├── NameLabel (Label, 12px)
│   ├── LevelLabel (Label, 11px)
│   └── ChainBadge (Label, "x1")
├── PartnerCardSlot_1
├── PartnerCardSlot_2
└── PartnerCardSlot_3
```

### run_main.gd 修改

```gdscript
## 在 _fill_partner_slot() 中：

func _fill_partner_slot(slot: Control, partner) -> void:
    ## slot 现在是一个 PanelContainer（小型卡片）
    var config_id: int = partner.get("partner_config_id", 0) if partner is Dictionary else partner.partner_config_id
    var cfg: Dictionary = ConfigManager.get_partner_config(config_id)
    
    ## 边框按稀有度
    var rarity: String = partner.get("rarity", "C") if partner is Dictionary else partner.rarity
    if rarity.is_empty():
        rarity = cfg.get("rarity", "C")
    var border_color: Color = _get_rarity_color(rarity)
    
    var style := StyleBoxFlat.new()
    style.bg_color = Color(0.06, 0.06, 0.08, 0.9)
    style.border_color = border_color
    style.border_width_left = 2
    style.border_width_top = 2
    style.border_width_right = 2
    style.border_width_bottom = 2
    style.corner_radius_top_left = 6
    style.corner_radius_top_right = 6
    style.corner_radius_bottom_left = 6
    style.corner_radius_bottom_right = 6
    slot.add_theme_stylebox_override("panel", style)
    
    ## 清空旧子节点（重新构建）
    for child in slot.get_children():
        child.queue_free()
    
    var vbox := VBoxContainer.new()
    vbox.alignment = BoxContainer.ALIGNMENT_CENTER
    vbox.add_theme_constant_override("separation", 4)
    slot.add_child(vbox)
    
    ## 头像（放大到 64x64，卡片感更强）
    var portrait := TextureRect.new()
    portrait.name = "Portrait"
    portrait.custom_minimum_size = Vector2(64, 64)
    portrait.expand_mode = TextureRect.EXPAND_FIT_WIDTH_PROPORTIONAL
    portrait.stretch_mode = TextureRect.STRETCH_KEEP_ASPECT_CENTERED
    var avatar_path: String = cfg.get("avatar_path", "")
    if not avatar_path.is_empty():
        portrait.texture = load(avatar_path)
    vbox.add_child(portrait)
    
    ## 名字
    var name_label := Label.new()
    name_label.name = "NameLabel"
    name_label.text = partner.get("name", "???") if partner is Dictionary else partner.name
    name_label.horizontal_alignment = HORIZONTAL_ALIGNMENT_CENTER

    level_label.add_theme_color_override("font_color", Color("#888888"))
    vbox.add_child(level_label)
    
    ## CHAIN 计数徽章
    var chain_count: int = partner.get("chain_count", 0) if partner is Dictionary else partner.chain_count
    var chain_badge := Label.new()
    chain_badge.name = "ChainBadge"
    chain_badge.text = "x chain %d" % chain_count
    chain_badge.horizontal_alignment = HORIZONTAL_ALIGNMENT_CENTER
    chain_badge.add_theme_font_size_override("font_size", 10)
    chain_badge.add_theme_color_override("font_color", Color("#5A8FD0"))
    vbox.add_child(chain_badge)
```

`_init_partner_slots()` 中创建 4 个 PanelContainer（替代原来的 HBoxContainer）：

```gdscript
func _init_partner_slots() -> void:
    for child in partner_panel.get_children():
        if child.name != "PartnerTitle":
            child.queue_free()
    _partner_slots.clear()
    
    for i in range(_max_partner_slots):
        var card := PanelContainer.new()
        card.name = "PartnerCardSlot_%d" % i
        card.custom_minimum_size = Vector2(140, 180)
        partner_panel.add_child(card)
        _partner_slots.append(card)
        card.visible = false
```

### 充能条处理

原来的 SkillGauge（ProgressBar）在小型卡片中放不下，有两种方案：

**方案 A（推荐）**：去掉 ProgressBar，用 CHAIN 计数 + 边框发光表示充能状态
- chain_count < max：正常边框
- chain_count >= max：边框变亮金并微微闪烁（已有 `_flash_partner_icon` 可复用逻辑）

**方案 B**：把充能条缩成细条放在卡片底部
- ProgressBar `custom_minimum_size = Vector2(120, 3)`
- 放在 ChainBadge 下方

建议用方案 A，更简洁。

## 素材需求

| 素材 | 用途 | 规格 |
|------|------|------|
| 伙伴 portrait（营救用） | RescuePopup 卡片中央大图 | 256x256 或 512x512 PNG，透明背景 |
| 伙伴 avatar（爬塔用） | RunMain PartnerHUD 小卡片头像 | 128x128 或 256x256 PNG，透明背景（可缩放） |

如果 portrait 和 avatar 是同一张图，只用 avatar 也行，RescuePopup 中放大显示即可。
