## 执行顺序建议

```
阶段1（P0数据层）：
  runtime_partner.gd → partner_configs.json → hero_configs.json
  → config_manager.gd → character_manager.gd

阶段2（P0回合结构）：
  node_pool_system.gd → node_resolver.gd → run_controller.gd
  → run_main.tscn/gd

阶段3（P1并行）：
  A: training_system.gd + attribute_mastery_configs.json
  B: battle_engine相关（damage_calculator + chain_trigger + partner_assist）
  C: shop_system.gd + shop_popup.gd
  D: pvp_director.gd + pvp_opponent_generator.gd

阶段4（P1终局）：
  settlement.gd + final_battle.gd + boss_configs.json

阶段5（P2收尾）：
  main_menu排行榜 + save_manager魔城币 + event_config.json
```
