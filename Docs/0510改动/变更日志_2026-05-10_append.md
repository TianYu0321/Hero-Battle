### 8. NodeResolver -- OUTING 事件系统更新后的适配

**文件**: `scripts/systems/node_resolver.gd`

**问题与修复**:
- `_OUTING_EVENTS` 已被更新为 v2 新格式（奖励类/惩罚类/精英类 3大类），但 `_resolve_outing()` 仍使用旧事件名 `"elite"` 和旧处理逻辑
- `elite` 检查条件错误: `selected_event == "elite"` 改为 `selected_event == "elite_battle"`（新事件名）
- `total_weight` 类型错误: `int` 改为 `float`，避免 `7.5` 等浮点权重被截断为 `7`
- `roll` 生成方式: `randi() % total_weight` 改为 `randf() * total_weight`，适配浮点权重
- 新增事件处理: `gold_bonus`（奖励20-50金币）、`full_heal`（恢复全部HP）、`thief`（损失20%金币标记）
- 移除已不存在的事件处理: `shop`、`rest`、`heal` 分支删除
