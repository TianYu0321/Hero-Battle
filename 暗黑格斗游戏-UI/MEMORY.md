## Current State
- 画布已锁定为 1920×1080，最新预览通过（376 nodes，0 error）。
- 顶部 HUD 已重排为中央竖向徽章塔：TIME → VS → ROUND → 场地名。
- 红蓝双侧采用完全镜像的网格结构（132px 头像 + 1fr 信息栏），尺寸严格对称。
- DESIGN.md 与 App.jsx 维持有效，CSS 冲突已清理。

## Artifacts
- App.jsx：主运行文件，1920×1080 锁定画布，三段式布局（HUD / 舞台 / 日志）。
- DESIGN.md：最小化设计系统文档，承载色彩、字体、圆角与间距规则（保持为设计基线）。

## Design Direction
- 暗黑哥特格斗电竞风格，参考传统 FTG 横向 HUD 排版。
- 中央徽章塔作为信息中轴，左右严格镜像呈现双方阵营。
- 高对比霓虹（红/蓝阵营光晕）配合暗金点缀，营造地下竞技场氛围。
- 信息层级：顶部实时状态 > 中段舞台焦点 > 底部历史与操作。

## User Feedback
- 明确要求暗黑风，必须包含顶部血条/名字/回合/VS 以及下方日志框架。
- 要求画布固定 1920×1080；VS、回合数、时间集中放置在中轴。
- 要求双方信息卡严格对称、同尺寸（已落地为镜像网格）。
- 历史修复（暴击数字重叠、底部按钮裁切）已确认通过。

## Decisions
- 画布尺寸锁定 1920×1080。
- 中央 HUD 采用四层徽章塔：TIME（红点闪烁）→ VS（红→金→蓝渐变 + 光晕脉动）→ ROUND（菱形旋转）→ 场地名分割线。
- 红蓝两侧使用 `132px + 1fr` 镜像网格，头像/HP/能量条/名字/段位徽章尺寸一致，仅切角与配色互为镜像。
- 三段式纵向布局：HUD → 舞台 → 日志面板。
- 血条分段刻度 + 顶部光泽渐变；菱形旋转回合徽标。
- 底部"跳过回合"以暗金边框区别于攻击/防御/技能。
- 字体栈：Cinzel（标题与数字）、Oxanium（数据指标）、Noto Sans SC（中文正文）。
